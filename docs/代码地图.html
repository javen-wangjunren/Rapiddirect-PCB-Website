PCB 项目代码地图

这是一份面向当前仓库的交互式代码地图，重点解释 `apps/ems-site` 如何把 Astro 前台、React 后台 CMS、Supabase 内容表、媒体库和联系表单 worker 串成一个完整系统。点击中间图上的节点，右侧会展示该部分的职责、关键文件、上下游关系和阅读建议。

前台读链路 页面运行时从 Supabase 拉取已发布内容，再与本地默认结构 merge + normalize 后渲染。
后台写链路 编辑器读写 `pages`、`page_content`、`seo_meta`，管理发布状态和 SEO 信息。
外围能力 媒体库走 Supabase Storage，询盘表单走独立 Cloudflare Worker + Resend。
